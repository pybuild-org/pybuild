# xml タグ

```xml
<xml></xml>
```

何の用途もないプレースホルダーで、スタックを使用せず、任意の回数、任意の階層で記述できる

# use タグ

```xml
<use file="" />
```

外部の xml 内容を取り込み、現在のコンテキストで直ちに実行する

`file` パラメータはファイルパス、または `http(s)://` で始まるリンクを指定できる

拡張子が `.xml` でない場合、`use` タグは自動的に付加する

# config タグ

```xml
<config [type="group"] name=""><config>
```

設定を宣言するタグで、`name` は設定名を宣言するために使用する

`type="group"` はオプションで、グループ設定を宣言する際に使用し、この場合その `config` タグは複数の子 `config` タグによって構成される

# prop タグ

```xml
<prop name="" value="" />
```

設定項目の具体的な値を宣言するタグで、`name` はフィールド名、`value` はフィールド値を宣言する

# run タグ

```xml
<run job="" | command="" />
```

タスクを実行するためのタグで、`job` と `command` はどちらか一方を選択

`job` はタスク名を宣言し、事前に設定されたタスクを実行するために使用する

`command` はコマンド内容を宣言し、システムコマンドを実行するために使用する

# log タグ

```xml
<log></log>
```

1 行のログを出力するために使用する
