# xml タグ

```xml
<xml></xml>
```

何の用途もないプレースホルダーで、スタックを占有せず、任意の回数、任意の階層で記述できます。

# define タグ

```xml
<define name="" value="" />
```

値エイリアスを定義するためのタグで、`name` はエイリアス名を宣言し、`value` は値を宣言します。

`value` パラメータは、以前の define タグで定義されたエイリアスを `{name}` で使用できます。

# use タグ

```xml
<use src="" />
```

外部の XML コンテンツを取り込み、現在のコンテキストで即座に実行します。

`src` パラメータはファイルパス、または `http(s)://` で始まるリンクのいずれかです。

`src` パラメータは、define タグで定義されたエイリアスを `{name}` で使用できます。

`.xml` 拡張子が付いていない場合、`use` タグは自動的に付加します。

# config タグ

```xml
<config [type="group"] name=""><config>
```

設定を宣言するためのタグで、`name` は設定名を宣言します。

`type="group"` はオプションで、グループ設定を宣言します。この場合、`config` タグは複数の子 `config` タグによって構成されます。

[汎用設定項目](./common-config.md)

# prop タグ

```xml
<prop name="" value="" />
```

設定項目の具体的な値を宣言するタグで、`name` はフィールド名、`value` はフィールド値を宣言します。

# run タグ

```xml
<run job="" | command="" />
```

タスクを実行するためのタグで、`job` と `command` はどちらか一方を選択します。

`job` はタスク名を宣言し、事前に設定されたタスクを実行します。

`command` はコマンド内容を宣言し、システムコマンドを実行します。

# log タグ

```xml
<log></log>
```

1 行のログを出力します。
