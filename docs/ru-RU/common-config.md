# Конфигурация тегов шаблона

Определение начального и конечного тегов, используемых шаблоном

```xml
<config name="template">
    <prop name="start-tag" value="" />
    <prop name="end-tag" value="" />
</config>
```

По умолчанию используется `{` в качестве начального тега и `}` в качестве конечного тега.
